Summary

We are going to implement the parallel version of Force Directed Graph – one of network visualization algorithms, using both OpenMP and MPI.

Background

In force-based layout systems, the graph drawing software modifies an initial vertex placement by continuously moving the vertices according to a system of forces based on physical metaphors related to systems of springs or molecular mechanics.

Challenge

The typical force-directed algorithms are generally considered to run in cubic time O(n^3), where n is the number of nodes of the input graph. This is because the number of iterations is estimated to be linear O(n), and in every iteration, all pairs of nodes need to be visited and their mutual repulsive forces computed. The problem has locality and a high computation ratio.
